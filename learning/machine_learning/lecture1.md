## 1. Machine learning definition
- Arthur Samuel (1959). Machine learning is a field of study that gives computers the ability to learn without being explicitly programmed.
- Tom Mitchell (1998). Well-posed learning problem: a computer program is said to learn from experience E with respect to some task T and some performance measure P, if its performance on T, as measured by P, improves with E
## 2. Supervised learning
- Supervised learning problem: given a dataset like 2D chart (inputs X and labels Y) to find the relationship mapping from X to Y. 
- Regression problem is that the value Y we're trying to predict is continuous.
- Classification problem is that the value Y takes on a discrete number of variables.
- Support Vector Machine allow using an infinite-dimensional vector to represent X.
